### _download

Download the [Community Edition](https://github.com/loouislow81/kraft.ui/releases/tag/4.6.22) binary for Linux.

### _prerequisites

`Kraft` requires `NodeJS` runtime to be installed,

```bash
$ sudo apt install -y nodejs
$ sudo npm i -g n
$ sudo n latest
```

### _run

```bash
$ npm i
$ npm start
```
